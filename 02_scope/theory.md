## Different Scopes in Ordered Priority
### Local
- Function Scope
- Applicable for variables defined in a function, class or block of code
- Can only be accessed within that function, class or block

### Enclosing
- Non-local Scope
- Variables defined in a nested function
- Can only be accessed from the code of the inner and enclosing functions

### Global
- Module Scope
- Variables and functions defined in the top or a program or module
- Can be accessed anywhere

### Built-In
- Applicable for variables and functions predefined by python
